## Secciones

- **Hero** — Typing effect, stats animados, botón de descarga de CV
- **About** — Bio, info personal, GitHub contribution graph
- **Habilidades** — Skills con barras de progreso e íconos
- **Proyectos** — 6 proyectos con tech stack y links a GitHub
- **Experiencia** — Timeline de carrera freelance
- **Educación** — Formación académica y logros
- **Contacto** — Formulario funcional (Formspree) + redes sociales

## Características técnicas

- Módulos ES nativos (`type="module"`)
- Scroll progress bar + reveal animations con IntersectionObserver
- Tema oscuro / Hacker Green con `localStorage`
- Cursor personalizado + partículas animadas
- Easter egg terminal (Konami: ↑↑↓↓←→←→BA)
- Formulario con Formspree — sin backend propio
- Email obfuscado contra bots de spam
- Schema JSON-LD para SEO estructurado
- Página 404 personalizada via `vercel.json`
- Totalmente responsive
